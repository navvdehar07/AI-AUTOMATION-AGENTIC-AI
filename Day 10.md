#  Day 10 – WhatsApp Chatbot Workflow

##  What is a WhatsApp Chatbot?

A WhatsApp chatbot is an automated system that can receive and reply to messages on WhatsApp without human intervention. It uses AI to understand user queries and provide instant responses, making communication faster and more efficient.

##  Tools Used

* **n8n** – To create and automate the workflow.
* **Evolution API** – To connect WhatsApp with n8n.
* **WhatsApp** – To send and receive messages.
* **AI Agent (Gemini/OpenAI)** – To generate intelligent responses.

##  Workflow Steps

1. Connect WhatsApp with Evolution API.
2. Link Evolution API with n8n.
3. Configure the Webhook to receive incoming messages.
4. Pass the user's message to the AI Agent.
5. Generate a response using AI.
6. Send the AI-generated response back to WhatsApp.
7. Test the chatbot by sending messages from WhatsApp.

##  What I Learned

* How WhatsApp chatbot automation works.
* How to integrate Evolution API with n8n.
* How webhooks receive incoming messages.
* How AI can generate automatic replies.
* How to test and troubleshoot a chatbot workflow.
