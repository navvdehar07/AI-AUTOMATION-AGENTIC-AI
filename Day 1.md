## Artificial Intelligence (AI)

Today was the first day of my AI Automation and Agentic AI journey.

I started by learning the basics of **Artificial Intelligence (AI)**. 

AI is technology that allows machines to think, learn, and make decisions like humans.

After that, I learned about **AI agents**. An AI agent is a system that can observe, think, and take actions to complete a task.

---

## Types of AI Agents

I studied different types of agents:

---

### 1. Simple Reflex Agent

This agent works based on current conditions only.

Example: A thermostat.

---

### 2. Model-Based Agent

This agent remembers past information and uses it for decisions.

---

### 3. Goal-Based Agent

This agent works to achieve a specific goal.

Example: Navigation apps.

---

### 4. Utility-Based Agent

This agent chooses the best option from many choices.

---

### 5. Learning Agent

This agent learns from experience and improves over time.

Example: Chatbots and recommendation systems.
